# Python-mundo1
Atividades feitas no mundo 1 de Python do curso em video, com foco na construção de uma base sólida em lógica de programação e nos fundamentos básicos da linguagem Python.
O objetivo deste projeto é registrar minha evolução, praticar conceitos essenciais e criar um histórico de aprendizado organizado.

# Conteúdos abordados:
Durante este módulo, trabalhei os seguintes conceitos:

-Introdução ao Python
-Entrada e saída de dados
-Variáveis e tipos primitivos
-Operadores aritméticos
-Manipulação de strings
-Condições
-Lógica de programação básica

# Organização do repositório
Os exercicíos estão organizados da seguinte forma:
  ex001.py (exercicío 1)
  ex002.py (exercicío 2)
  ex003.py (exercicío 3)
Cada arquivo corresponde a um exercicío proposto no curso

# Tecnologias utilizadas:
-Python
-Pycharm

-Fixar os fundamentos do Python
-Desenvolver raciocínio lógico
-Criar consistência nos estudos
-Iniciar a construção de um portfólio em programação
